SLIDE 5: TECHNOLOGY OVERVIEW
================================================================================

TITLE: Technology Behind KnowMe

KEY TECHNOLOGIES:
┌─────────────────────────────────────────────┐
│  Embeddings + Vector Database + LLM = RAG  │
└─────────────────────────────────────────────┘

EXPLAINED SIMPLY:
• Embeddings: Convert text to numbers AI understands
• Vector DB: Store and search by meaning (not keywords)
• LLM (GPT-4): Generate human-like responses
• RAG: Retrieval-Augmented Generation

WHY THIS MATTERS:
"Traditional search finds 'React' only if you type 'React'
Vector search understands 'frontend framework' means React too!"
