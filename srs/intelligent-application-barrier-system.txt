1. PROBLEM STATEMENT
================================================================================

THE APPLICATION TSUNAMI
-----------------------
LinkedIn processes 11,000+ applications per minute with a 45% surge in the past
year. AI tools have created an "applicant tsunami" where:

- 40% of job seekers use AI to write/refine resumes
- 26% use AI for mass-applying to jobs
- 73% of employers receive AI-generated resumes with false information
- 65% report a surge in underqualified candidates
- HR spends 9.24 days average sorting through irrelevant applications per role

THE CORE PROBLEM
----------------
For every job opening with 5 positions:
- 1000+ applications flood in
- 80%+ are low-quality or mismatched
- HR wastes time screening unqualified candidates
- Qualified candidates get lost in the noise
- AI screening creates additional false positives/negatives

This creates a "Doom Loop":
+----------------+      +------------------+      +-------------------+
| Candidates     | ---> | Mass Apply       | ---> | HR Overwhelmed    |
| Can't Stand    |      | Using AI Tools   |      | Can't Review All  |
| Out            |      |                  |      |                   |
+----------------+      +------------------+      +-------------------+
        ^                                                  |
        |                                                  v
        +---------- Candidates Use MORE AI Tools <---------+

2. CURRENT MARKET SOLUTIONS & THEIR LIMITATIONS
================================================================================

SOLUTION 1: HARD GATES (Score-Based Blocking)
---------------------------------------------
How it works: Block applications if match score < threshold
Limitations:
- Prevents passionate candidates from applying
- Ignores potential and learning ability
- Creates unfair barriers for career changers
- Students/fresh graduates suffer the most

SOLUTION 2: AI SCREENING (Greenhouse Real Talent, etc.)
-------------------------------------------------------
How it works: AI filters applications into quality tiers
Limitations:
- Still processes thousands of applications
- Can have bias issues
- Doesn't change candidate behavior
- HR still reviews large volumes

SOLUTION 3: FRAUD DETECTION (Ashby, SignalSyft)
-----------------------------------------------
How it works: Detect bot submissions, fake profiles
Limitations:
- Only addresses fraud, not volume
- Doesn't help genuine mismatched candidates
- Reactive rather than proactive

SOLUTION 4: PREMIUM APPLICATIONS (LinkedIn's "Dream Job")
---------------------------------------------------------
How it works: 1 premium application per month highlighted to recruiters
Limitations:
- Creates pay-to-win dynamics
- Doesn't address root cause
- Single designation is too restrictive

3. OUR INNOVATIVE SOLUTION: IABS
   (Intelligent Application Barrier System)
================================================================================

CORE PHILOSOPHY
---------------
"Don't block anyone, but make them earn their application through understanding"

Instead of stopping candidates, we create "intelligent friction" that:
1. Educates candidates about their fit
2. Filters out low-intent applications naturally
3. Ensures only serious candidates reach HR
4. Provides value to candidates even when they don't apply

SWIPE ACTIONS & CONSEQUENCES
----------------------------
┌──────────────────┬──────────────────────────────────────────────────────────┐
│ ACTION           │ RESULT                                                   │
├──────────────────┼──────────────────────────────────────────────────────────┤
│ Swipe Left       │ Skip job, won't show again (unless filtered differently)│
│ Swipe Right      │ Opens Stage 2: Reality Check modal                      │
│ Swipe Up         │ Super Interest - Priority application flag              │
│ Tap Save (⭐)    │ Saves for later review without expressing interest      │
│ Tap Details      │ Opens full job page for reading                         │
└──────────────────┴──────────────────────────────────────────────────────────┘

HOW SWIPE MODE HELPS SOLVE THE PROBLEM
--------------------------------------
1. REDUCES MASS APPLICATIONS
   - Each swipe is deliberate
   - No "Apply All" buttons
   - Natural pace limits applications

2. INCREASES ENGAGEMENT
   - Gamified experience
   - Less overwhelming than lists
   - Better mobile experience

3. CREATES MEANINGFUL DATA
   - Swipe patterns reveal preferences
   - Better recommendations over time
   - Helps HR understand candidate interests

4. FILTERS INTENT
   - Swiping right = Initial interest checkpoint
   - Forces brief review of each job
   - Skip rate reveals true market interest
