1. EXECUTIVE SUMMARY & KEY CHANGES FROM V1
================================================================================

The AI-Verified Learning Goals System V2 represents a complete reimagining of
the verification process, leveraging existing platform components (Studio, Quiz,
CodeEditor, Voice) to create a seamless, integrated learning experience.

KEY CHANGES FROM V1:
────────────────────────────────────────────────────────────────────────────────

V1 APPROACH (DEPRECATED)                   V2 APPROACH (NEW)
─────────────────────────                  ──────────────────────
User uploads documents/screenshots         User completes AI-generated tasks
Manual proof submission required           Automatic verification through tasks
Separate verification interface            Integrated Studio-based learning
Single verification method                 Tab-based multi-type verification
Generic assignment generation              OpenAI Assistants with schema format
Complex goal creation form                 Simple click-based goal creation
Separate notes/learning area              Studio integration for notes/practice

WHY V2 IS BETTER:
────────────────────────────────────────────────────────────────────────────────
• NO manual uploads - verification happens through task completion
• REUSES existing platform components (Studio, Quiz, CodeEditor, Voice)
• SIMPLER goal creation - click-click-click, no long descriptions
• INTEGRATED learning experience - practice and verify in one place
• DYNAMIC verification - only shows relevant question types
• SERVER-SIDE rendering with client components for optimal performance

5. AI ASSIGNMENT GENERATION (OPENAI ASSISTANTS)
================================================================================

The assignment generation uses OpenAI Assistants API with a structured schema,
similar to how projects are generated in @apps/main/app/(main)/projects/.

5.1 OPENAI ASSISTANT CONFIGURATION
────────────────────────────────────────────────────────────────────────────────

Assistant Name: learning-goals-assignment-generator
Model: gpt-4o or gpt-4-turbo-preview

System Instructions:
```
You are an AI learning assignment generator for CoderzLab. Your job is to create
comprehensive, level-appropriate learning assignments based on a user's learning
goal.

Based on the goal title, category, level, and focus areas, generate assignments
in the following types:
- QUIZ: Multiple choice questions testing Learnual understanding
- CODING: Programming problems to test practical skills
- MOCK: Interview questions for voice-based assessment
- PROJECT: Only for applicable topics (web dev, etc.) - NOT for pure DSA

IMPORTANT RULES:
1. DSA/Algorithm goals should NOT include PROJECT type
2. Web Dev/Frontend/Backend goals SHOULD include PROJECT type
3. All goals should have at least QUIZ
4. Level affects difficulty and depth
5. Focus areas guide emphasis

Return your response in the EXACT JSON schema provided.
```

const goalData = await getGoalWithAssignments(goalId);

// Pass to client component

8.1 NEW PRISMA MODELS

```prisma
// ==================== LEARNING GOALS ====================

enum LearningGoalCategory {
    DSA
    WEB_DEVELOPMENT
    BACKEND
    FRONTEND
    DEVOPS
    AI_ML
    DATABASE
    SYSTEM_DESIGN
    OTHER
}

enum LearningGoalLevel {
    BEGINNER
    INTERMEDIATE
    ADVANCED
    EXPERT
}

enum LearningGoalStatus {
    ACTIVE
    COMPLETED
    VERIFIED
    ABANDONED
}

enum VerificationStatus {
    PENDING
    IN_PROGRESS
    COMPLETED
    PASSED
    FAILED
}

model LearningGoal {
    id          String              @id @default(uuid())
    userId      String
    user        User                @relation("UserLearningGoals", fields: [userId], references: [id], onDelete: Cascade)
    
    // Basic Info
    title       String
    category    LearningGoalCategory
    level       LearningGoalLevel
    focusAreas  String[]            // Array of focus areas
    targetDate  DateTime?
    
    // Status
    status      LearningGoalStatus  @default(ACTIVE)
    progress    Int                 @default(0) // 0-100
    
    // Timestamps
    createdAt   DateTime            @default(now())
    updatedAt   DateTime            @updatedAt
    completedAt DateTime?
    verifiedAt  DateTime?
    
    // Relations
    assignments     LearningAssignment?
    verification    GoalVerification?
    studioContent   LearningGoalStudio?
    
    @@index([userId])
    @@index([status])
    @@index([category])
}

model LearningAssignment {
    id          String          @id @default(uuid())
    goalId      String          @unique
    goal        LearningGoal    @relation(fields: [goalId], references: [id], onDelete: Cascade)
    
    // Assignment Types (dynamic - only types that exist)
    assignmentTypes String[]    // ["quiz", "coding", "mock", "project"]
    
    // Raw AI Response (for debugging/auditing)
    rawResponse     Json?
    
    // Individual Assignments (stored as JSON for flexibility)
    quizData        Json?       // QuizAssignment structure
    codingData      Json?       // CodingAssignment structure
    mockData        Json?       // MockAssignment structure
    projectData     Json?       // ProjectAssignment structure
    
    // Timestamps
    generatedAt     DateTime    @default(now())
    
    @@index([goalId])
}

model GoalVerification {
    id              String              @id @default(uuid())
    goalId          String              @unique
    goal            LearningGoal        @relation(fields: [goalId], references: [id], onDelete: Cascade)
    
    // Overall Status
    status          VerificationStatus  @default(PENDING)
    overallScore    Int?                // 0-100
    
    // Individual Section Scores
    quizScore       Int?
    codingScore     Int?
    mockScore       Int?
    projectComplete Boolean             @default(false)
    
    // Individual Section Status
    quizStatus      VerificationStatus  @default(PENDING)
    codingStatus    VerificationStatus  @default(PENDING)
    mockStatus      VerificationStatus  @default(PENDING)
    projectStatus   VerificationStatus  @default(PENDING)
    
    // Detailed Results (JSON for flexibility)
    quizResults     Json?
    codingResults   Json?
    mockResults     Json?
    projectResults  Json?
    
    // Timestamps
    startedAt       DateTime?
    completedAt     DateTime?
    
    // Related Records
    quizAttempts        GoalQuizAttempt[]
    codingSubmissions   GoalCodingSubmission[]
    mockSessions        GoalMockSession[]
    
    @@index([goalId])
    @@index([status])
}

model GoalQuizAttempt {
    id              String              @id @default(uuid())
    verificationId  String
    verification    GoalVerification    @relation(fields: [verificationId], references: [id], onDelete: Cascade)
    userId          String
    
    // Results
    score           Int                 // 0-100
    correctCount    Int
    totalQuestions  Int
    timeTaken       Int                 // in seconds
    answers         Json                // Array of answers with questionId, selected, isCorrect
    
    // Timestamps
    startedAt       DateTime
    completedAt     DateTime            @default(now())
    
    @@index([verificationId])
    @@index([userId])
}

model GoalCodingSubmission {
    id              String              @id @default(uuid())
    verificationId  String
    verification    GoalVerification    @relation(fields: [verificationId], references: [id], onDelete: Cascade)
    userId          String
    
    // Problem Info
    problemId       String
    problemTitle    String
    
    // Submission
    code            String
    language        String
    
    // Results
    passed          Boolean
    testResults     Json?               // Array of test case results
    
    // Timestamps
    submittedAt     DateTime            @default(now())
    
    @@index([verificationId])
    @@index([userId])
    @@index([problemId])
}

model GoalMockSession {
    id              String              @id @default(uuid())
    verificationId  String
    verification    GoalVerification    @relation(fields: [verificationId], references: [id], onDelete: Cascade)
    userId          String
    
    // ElevenLabs Data
    conversationId  String?
    
    // Results
    score           Int?                // 0-100
    feedback        Json?               // AI-generated feedback
    transcript      Json?               // Conversation transcript
    
    // Duration
    duration        Int?                // in seconds
    
    // Timestamps
    startedAt       DateTime            @default(now())
```

```typescript
'use server'

import { prisma } from '@repo/prisma'
import { auth } from '@/auth'
import { revalidatePath } from 'next/cache'
import { generateAssignmentsWithAI } from './assignments.action'

interface CreateGoalInput {
    title: string
    category: string
    level: string
    targetDate?: Date
    focusAreas?: string[]
}

export async function createGoal(input: CreateGoalInput) {
    const session = await auth()
    if (!session?.user?.id) {
        return { success: false, error: 'Unauthorized' }
    }

    try {
        // 1. Create the learning goal
        const goal = await prisma.learningGoal.create({
            data: {
                userId: session.user.id,
                title: input.title,
                category: input.category,
                level: input.level,
                targetDate: input.targetDate,
                focusAreas: input.focusAreas || [],
            }
        })

        // 2. Generate assignments using OpenAI Assistants
        const assignmentResult = await generateAssignmentsWithAI({
            goalId: goal.id,
            title: input.title,
            category: input.category,
            level: input.level,
            focusAreas: input.focusAreas || [],
        })

        if (!assignmentResult.success) {
            // Rollback goal creation if assignment generation fails
            await prisma.learningGoal.delete({ where: { id: goal.id } })
            return { success: false, error: 'Failed to generate assignments' }
        }

        // 3. Create studio content for the goal
        await prisma.learningGoalStudio.create({
            data: {
                goalId: goal.id,
                content: { blocks: [] }
            }
        })

        revalidatePath('/goals')
        return { success: true, goalId: goal.id }

    } catch (error) {
        console.error('Error creating goal:', error)
        return { success: false, error: 'Failed to create goal' }
    }
}

export async function getGoals() {
    const session = await auth()
    if (!session?.user?.id) {
        return { success: false, error: 'Unauthorized' }
    }

    try {
        const goals = await prisma.learningGoal.findMany({
            where: { userId: session.user.id },
            include: {
                assignments: true,
                verification: true,
            },
            orderBy: { createdAt: 'desc' }
        })

        return { success: true, goals }
    } catch (error) {
        console.error('Error fetching goals:', error)
        return { success: false, error: 'Failed to fetch goals' }
    }
}

export async function getGoalWithAssignments(goalId: string) {
    const session = await auth()
    if (!session?.user?.id) {
        return { success: false, error: 'Unauthorized' }
    }

    try {
        const goal = await prisma.learningGoal.findUnique({
            where: { id: goalId, userId: session.user.id },
            include: {
                assignments: true,
                verification: true,
                studioContent: {
                    include: {
                        chatMessages: {
                            orderBy: { createdAt: 'asc' }
                        }
                    }
                }
            }
        })

        if (!goal) {

        }

        return { success: true, goal }
```

```typescript
'use server'

import OpenAI from 'openai'
import { prisma } from '@repo/prisma'

const openai = new OpenAI({
    apiKey: process.env.OPENAI_API_KEY
})

interface GenerateAssignmentsInput {
    goalId: string
    title: string
    category: string
    level: string
    focusAreas: string[]
}

export async function generateAssignmentsWithAI(input: GenerateAssignmentsInput) {
    try {
        // Use OpenAI Assistants API (like projects module)
        const thread = await openai.beta.threads.create()

        await openai.beta.threads.messages.create(thread.id, {
            role: 'user',
            content: JSON.stringify({
                goal: {
                    title: input.title,
                    category: input.category,
                    level: input.level,
                    focusAreas: input.focusAreas
                }
            })
        })

        const run = await openai.beta.threads.runs.create(thread.id, {
            assistant_id: process.env.LEARNING_GOALS_ASSISTANT_ID!,
        })

        // Poll for completion
        let runStatus = await openai.beta.threads.runs.retrieve(thread.id, run.id)
        while (runStatus.status !== 'completed') {
            if (runStatus.status === 'failed') {
                throw new Error('Assistant run failed')
            }
            await new Promise(resolve => setTimeout(resolve, 1000))
            runStatus = await openai.beta.threads.runs.retrieve(thread.id, run.id)
        }

        // Get the response
        const messages = await openai.beta.threads.messages.list(thread.id)
        const assistantMessage = messages.data.find(m => m.role === 'assistant')
        
        if (!assistantMessage || assistantMessage.content[0].type !== 'text') {
            throw new Error('Invalid assistant response')
        }

        // Parse JSON from response
        const responseText = assistantMessage.content[0].text.value
        const jsonMatch = responseText.match(/```json\n?([\s\S]*?)\n?```/)
        const assignments = JSON.parse(jsonMatch ? jsonMatch[1] : responseText)

        // Save to database
        await prisma.learningAssignment.create({
            data: {
                goalId: input.goalId,
                assignmentTypes: assignments.assignmentTypes,
                rawResponse: assignments,
                quizData: assignments.quiz,
                codingData: assignments.coding,
                mockData: assignments.mock,
                projectData: assignments.project,
            }
        })

        // Create verification record
        await prisma.goalVerification.create({
            data: {
                goalId: input.goalId,
                status: 'PENDING',
                quizStatus: assignments.quiz ? 'PENDING' : 'COMPLETED',
                codingStatus: assignments.coding ? 'PENDING' : 'COMPLETED',
                mockStatus: assignments.mock ? 'PENDING' : 'COMPLETED',
                projectStatus: assignments.project ? 'PENDING' : 'COMPLETED',
            }
        })

        return { success: true, assignments }

    } catch (error) {
        console.error('Error generating assignments:', error)
        return { success: false, error: 'Failed to generate assignments' }
    }
}
```

```typescript
'use server'

import { prisma } from '@repo/prisma'

// Submit Quiz Results
export async function submitQuizVerification(
    goalId: string,
    results: {
        score: number
        correctCount: number
        totalQuestions: number
        timeTaken: number
        answers: any[]
    }
) {
    const session = await auth()
    if (!session?.user?.id) {
        return { success: false, error: 'Unauthorized' }
    }

    try {
        const verification = await prisma.goalVerification.findUnique({
            where: { goalId }
        })

        if (!verification) {
            return { success: false, error: 'Verification not found' }
        }

        // Create quiz attempt record
        await prisma.goalQuizAttempt.create({
            data: {
                verificationId: verification.id,
                userId: session.user.id,
                score: results.score,
                correctCount: results.correctCount,
                totalQuestions: results.totalQuestions,
                timeTaken: results.timeTaken,
                answers: results.answers,
                startedAt: new Date(Date.now() - results.timeTaken * 1000),
            }
        })

        // Update verification status
        const passed = results.score >= 75
        await prisma.goalVerification.update({
            where: { id: verification.id },
            data: {
                quizScore: results.score,
                quizStatus: passed ? 'PASSED' : 'FAILED',
                quizResults: results,
            }
        })

        await checkAndUpdateOverallVerification(goalId)
        revalidatePath(`/goals/${goalId}/verify`)

        return { success: true, passed }

    } catch (error) {
        console.error('Error submitting quiz:', error)
        return { success: false, error: 'Failed to submit quiz' }
    }
}

// Check if all sections are complete and update overall status
async function checkAndUpdateOverallVerification(goalId: string) {
    const verification = await prisma.goalVerification.findUnique({
        where: { goalId }
    })

    if (!verification) return

    const allComplete = 
        verification.quizStatus !== 'PENDING' &&
        verification.codingStatus !== 'PENDING' &&
        verification.mockStatus !== 'PENDING' &&
        verification.projectStatus !== 'PENDING'

    if (allComplete) {
        const allPassed = 
            (verification.quizStatus === 'PASSED' || verification.quizStatus === 'COMPLETED') &&
            (verification.codingStatus === 'PASSED' || verification.codingStatus === 'COMPLETED') &&
            (verification.mockStatus === 'PASSED' || verification.mockStatus === 'COMPLETED') &&
            (verification.projectStatus === 'PASSED' || verification.projectStatus === 'COMPLETED')

        // Calculate overall score
        const scores = [
            verification.quizScore,
            verification.codingScore,
            verification.mockScore,
        ].filter(s => s !== null) as number[]
        
        const overallScore = scores.length > 0 
            ? Math.round(scores.reduce((a, b) => a + b, 0) / scores.length)
            : 0

        await prisma.goalVerification.update({
            where: { goalId },
            data: {
                status: allPassed ? 'PASSED' : 'FAILED',
                overallScore,
                completedAt: new Date(),
            }
        })

        // Update learning goal status
        await prisma.learningGoal.update({
            where: { id: goalId },
            data: {
                status: allPassed ? 'VERIFIED' : 'COMPLETED',
                verifiedAt: allPassed ? new Date() : null,
                completedAt: new Date(),
            }
        })
    }
}
```
